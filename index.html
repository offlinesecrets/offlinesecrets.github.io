Store secrets offline

Encryption tools based on AES and Shamir algorithms that split secrets into multiple parts. This allows to securely store those secrets offline (i.e. in a pgysical form in the real world) by placing the parts in different locations. Use one of the following options.

1. Simple encryption

Use AES algorithm to encrypt secret and generate QR code. More info
Provide: your secret and password  |  Store: QR code with the encrypted secret and the password  |  Recover: use password and QR code

Offline storage

You can store the secret securely offline by placing the password and the QR code in two different locations.

2. Split secret into multiple shares

Use Shamir algorithm to split secret into shares. The secret can be recovered using the subset of the generated shares. More info
Provide: your secret  |  Store: secret divided into multiple shares  |  Recover: use subset of shares

Offline storage

You can store the secret securely offline by placing the shares in multiple different locations.

3. Encrypt secret with password split into multiple shares

Use AES and Shamir algorithms to encrypt & split secret into shares. The secret can be recovered using the subset of the generated shares. More info
Provide: your secret  |  Store: QR code with the encrypted secret and the password split into multiple shares  |  Recover: use the subset of shares and the QR code

Offline storage

You can store the secret securely offline by placing the shares in multiple different locations and with the QR code placed only in some of the locations.
